# TchouTchou5G

App web mobile pour cartographier la qualité du réseau internet le long d'un trajet en train.

## Mise en route

1. Exécuter `schema.sql` dans le SQL Editor du projet Supabase (une seule fois).
2. Ouvrir l'app (servie en HTTPS via GitHub Pages), créer un compte, puis démarrer un enregistrement en début de trajet.
3. Garder l'écran allumé et l'onglet au premier plan pendant tout le trajet.

## Réglages

Intervalle entre pings, timeout, taille de la fenêtre glissante et seuils de couleur sont réglables depuis l'écran "Réglages" de l'app.
